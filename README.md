## 💰 如何记录交易

当你买入股票时，在 `transactions.csv` 中添加一行：

```csv
2026-05-06,601088,中国神华,BUY,32.50,184,5980.00,第一阶段建仓
```

字段说明：
- date: 交易日期 (YYYY-MM-DD)
- stock_code: 股票代码
- stock_name: 股票名称
- action: 操作类型 (BUY/SELL)
- price: 成交价格
- quantity: 成交数量
- amount: 成交金额
- notes: 备注

## 💵 如何记录分红

当收到分红时，在 `dividends.csv` 中添加一行：

```csv
2026-07-15,601088,中国神华,2.50,184,460.00,46.00,414.00,2025年度分红
```

## ⚠️ 投资原则

1. **不追高**：价格超过"谨慎买点"时不买
2. **分批建仓**：不要一次性满仓
3. **分红再投**：将分红继续买入对应股票
4. **长期持有**：建议持有3年以上
