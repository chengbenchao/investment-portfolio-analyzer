## 📝 网站使用说明

### 添加股票步骤
1. 打开网站
2. 在"输入股票名称或代码"框中输入（如：贵州茅台）
3. 选择行业（可选）
4. 点击"➕ 添加股票"按钮
5. 自动完成！

### 支持的输入格式
- **股票名称**：贵州茅台、招商银行
- **股票代码**：600519、000001
- **部分匹配**：茅台、招商

## 🎯 估值方法说明

### 买点计算逻辑
1. 取行业历史均值PE（如白酒=35x）
2. 计算打折系数：
   - 理想买点 = 50% × 行业均值
   - 合理买点 = 70% × 行业均值
   - 谨慎买点 = 85% × 行业均值
3. 结合当前PE，计算目标价格

### 数据标签说明
- 🟢 **实时** - 腾讯财经API获取
- 🟡 **估算** - 基于实时数据计算
- 🔵 **参考** - 行业均值或历史数据
